# Linguist AI APP Built with AI Studio

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

4. Set the `QWEN_API_KEY` in [.env.local](.env.local) to your Qwen API key

## Features 

- Generate French learning content based on a topic and difficulty level
- View the English translation
- Practice vocabulary with multiple choice questions
- [] Listen to the generated French text
